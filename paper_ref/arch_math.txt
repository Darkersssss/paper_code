370: tation), where each layer’s input is the output of the previous
371: one (i.e., hq = hk,v = hn). We apply a graph’s mask Mt,

379: nections, enhancing path-finding. This enables the encoder

566: Critic decoder: We developed our critic network, based
567: on [33], to selectively extract pertinent information for each
568: agent by incorporating other agents’ actions. The critic
569: network Qϕ, parameterized by ϕ, utilizes the same encoder
570: structure as the policy network πθ but features a distinct
571: decoder. During training, the critic receives full ground-
572: truth information, such as the entire map or graph (see
573: Fig. 2), enhancing training stability through privileged learn-
574: ing. Using the enhanced node features ˜hn from the encoder,
575: we retrieve the features of other agents, ˜hn
576: ψ−i(t), and their
577: corresponding features at the next timestep, ˜hn
578: ψ−i(t+1), based
579: on their current actions at. These key-value pairs augment
580: the current node feature hc with other agents’ actions, aiding
581: credit assignment.. The enhanced current node features ˜hc
582: are combined with enhanced neighboring features ˜hn
583: e , com-
584: puted in the same manner as in the policy decoder. Finally,
585: these concatenated features are mapped to state-action values
586: Qϕ(st, at) using a feed-forward layer. The networks are
587: trained using soft actor-critic (SAC) algorithm [34].
588: Training details: We use a training dataset of 5,663
589: randomly generated large-scale maps (see Fig. 3) and an
590: additional 100 unseen maps for testing. Each environment
591: measures 90m by 90m with 4 agents, a sensor range of
592: ds = 10m, and a 120◦FoV. Training parameters include
593: a maximum episode length of 128 steps, a discount factor
594: of γ = 1, a batch size of 256, and an episode buffer size of
595: 10,000. Training begins after 2,000 steps, with target entropy
596: 0.01·log(k), one update per step, and Adam optimizer (10−5
597: LR). The target critic updates every 256 steps. Complete
598: code and model are available at https://github.com/
599: marmotlab/MARVEL.
